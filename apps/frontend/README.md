# Getting Started


  You'll need an API key from https://www.weatherapi.com/ to access the latest weather info.


### Installation



&nbsp;

Super easy, first run this:



`yarn dev`




Then to run:



    yarn dev



That's it!



Now start using it with the functions and documentation below.
